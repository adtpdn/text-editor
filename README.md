# Web Code Editor

A lightweight browser-based code editor with GitHub integration and VS Code-like features.

## Features

- 📂 File Management (Open, Save, Multiple Tabs)
- 🎨 Syntax Highlighting
- 🔍 Code Outline/Navigator
- 📦 GitHub Integration
- 🌗 Dark Theme
- 📱 Responsive Design

## Supported Languages

- JavaScript/TypeScript
- HTML/XML
- CSS
- Python
- Ruby
- PHP
- Java
- C/C++/C#
- JSON
- Markdown
- Plain Text

## Quick Start

1. Open `index.html` in your browser
2. For GitHub features:
   - Get a GitHub token (Settings → Developer settings → Personal access tokens → Tokens)
   - Login with your token in the editor

## Dependencies

- CodeMirror 5.65.2
- Font Awesome 5.15.4

## Usage Tips

### Basic Operations
- New File: Click "New"
- Open File: Click "Open File"
- Save: Click "Save"
- Change Language: Use syntax dropdown

### GitHub Features
- Login with token
- Clone repositories
- Commit & push changes
- Pull updates
